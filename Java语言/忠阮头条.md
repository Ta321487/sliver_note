# 编辑网络

>虚拟机固定IP200，需要先改网段为200

1. 编辑→虚拟网络编辑器，选中NAT
![](assets/忠阮头条/file-20260123091437984.png)

2. 指定该虚拟机的网络
右键该虚拟机→设置→选择网络适配器→选择自定义→下拉列表选择NAT8

![](assets/忠阮头条/file-20260123091641088.png)

3. 远程连接该虚拟机
Finalshell下载：https://www.hostbuf.com/t/988.html
